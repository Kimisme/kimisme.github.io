引入依赖环境

编写启动类

编写控制层

  • @RestController等同于@Controller和@ResponseBody的组合
  • @RequestMapping用来配置url映射
  • @RequestParam用来匹配请求参数，可带参数public String index(@RequestParam(value="name",required=true,defaultValue="kim") String username)
  • @PathVariable用来匹配请求路径，可带参数public String index(@PathVariable(value="id",required=true) Integer myId)

组合注解

什么是组合注解，就是把省略了请求方式，如：
@RequestMapping(value="/get/{id}",method=RequestMethod.GET)等价与@GetMapping(value="/get/{id}")

其他组合注解

  • @PostMapping
  • @DeleteMapping
  • @PutMapping
