Log4j的构成

Log4j由三个重要组件构成：日志信息的优先级，日志信息的输出目的地，日志信息的输出格式。
日志信息的优先级用来指定这条日志信息的重要程度，从高到低有：ERROR，WARN，INFO，DEBUG；
日志信息的输出目的地指定了日志将打印到控制台还是文件中；
日志的输出格式控制了日志信息的显示内容。
