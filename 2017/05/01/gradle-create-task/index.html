Analyze

  • Task除了执行操作之外，还可以包含多个Property
  • Gradle为每个Task默认定义的Property有description，logger等
  • 每一个特定的Task类型可以含有特定的Property，比如Copy的from和to
  • 我们也可以动态地向Task中加入额外的Property
  • Gradle在执行Task时分为两个阶段：首先是配置阶段，然后才是实际执行阶段。在执行hello8之前，Gradle会扫描整个build.gradle文档，将hello8的description设置为”this is hello8”，然后执行hello8
